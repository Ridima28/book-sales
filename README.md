# 📚 Book Sales Data Analysis

## 📖 Project Overview

This project focuses on analyzing a Book Sales dataset obtained from Kaggle to uncover valuable insights into sales performance, customer genre, and book name. Through data cleaning, exploratory data analysis (EDA), and visualization, the project aims to identify key factors that influence book sales and support data-driven business decisions.

---

## 🎯 Objectives

* Analyze overall book sales performance.
* Identify top-selling books, authors, and genres.
* Discover sales trends over time.
* Understand customer purchasing behavior.
* Generate actionable insights through data visualization.

---

## 📂 Dataset

* **Source:** Kaggle
* **Dataset Type:** Book Sales Data
* **Format:** CSV

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Project Workflow

### 1. Data Collection

* Imported the dataset from Kaggle.
* Loaded data into a Jupyter Notebook environment.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Checked for inconsistencies and outliers.

### 3. Exploratory Data Analysis (EDA)

* Examined dataset structure and summary statistics.
* Analyzed book sales distribution.
* Explored relationships between different variables.
* Investigated trends across genres, authors, and time periods.

### 4. Data Visualization

Created visualizations to better understand:

* Sales distribution
* Top-selling books
* Best-performing authors
* Genre-wise sales performance
* Revenue trends over time

## 📊 Key Insights

* Identified the highest-selling books and authors.
* Discovered genres contributing the most to overall sales.
* Observed trends in customer demand across different periods.
* Found patterns that can help optimize inventory and marketing strategies.
* Highlighted revenue-driving segments within the book market.
